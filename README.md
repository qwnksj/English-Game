# 英语单词记忆游戏 - 网页版

🎮 一个基于Web的英语单词记忆游戏，专为安徽专升本学生设计，包含3500+核心词汇。

## 🌟 功能特性

### 🎯 核心功能
- **完全随机测试**: 从词库中随机抽取单词进行测试
- **自定义练习**: 个性化设置单词数量、难度和选项
- **复习错题**: 重点复习答错的单词
- **单元复习**: 按单元分组系统学习
- **学习统计**: 详细的学习进度和成绩分析

### 📚 词库管理
- 7个预设词库，总计3500+单词
- 支持自定义词库导入
- 词库转换工具（TXT → JSON）
- 词库合并与去重

### 📊 学习统计
- 学习进度实时跟踪
- 正确率统计分析
- 错题集自动生成
- 游戏记录保存

## 🚀 快速开始

### 本地运行
1. 下载所有文件到本地
2. 将单词文件放在 `词库文件/` 文件夹中
3. 用浏览器打开 `index.html`

### 在线部署
1. Fork 本仓库到你的GitHub账户
2. 在Settings → Pages中启用GitHub Pages
3. 选择主分支作为源
4. 访问 `https://[你的用户名].github.io/word-game`
